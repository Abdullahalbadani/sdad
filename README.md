# SADAD COM LIMITED Website

موقع تعريفي احترافي ثنائي اللغة لشركة **سداد كوم المحدودة**.

## التشغيل

```bash
npm install
npm run dev
```

## الفحص والبناء

```bash
npm run lint
npm run build
```

## الملفات الرسمية

- `public/profile-ar.pdf` الملف التعريفي العربي.
- `public/profile-en.pdf` الملف التعريفي الإنجليزي.
- `public/images/logo-symbol.jpg` الشعار الجديد.
- `public/images/logo-full.jpg` الشعار الكامل.

## ملاحظات

- الموقع يدعم العربية RTL والإنجليزية LTR من زر عائم.
- الروابط تعمل عبر Hash Routing لتسهيل الرفع على الاستضافات الثابتة.
- نموذج التواصل يفتح واتساب برسالة منظمة تحتوي بيانات الطلب.
